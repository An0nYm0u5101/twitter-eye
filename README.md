# Twitter-eye

Twitter Information Gathering Tool

![Screenshot](img/banner.png)

## Install Twitter Eye:
* git clone https://github.com/BullsEye0/twitter-eye.git

* cd twitter-eye
  
* pip3 install -r requirements.txt

  
## Use:
* python3 twitter-eye.py
